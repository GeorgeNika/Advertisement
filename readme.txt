Доска объявлений
=============================

Данное приложение можно использовать
для публикации, просмотра и хранении объявлений

=============================

Настройки приложения находятся в файле
main/resources/config.properties

path.application_log  - путь для хранение текущих логов
path.application_archive_log  - путь для хранение архивов логов

db.driver - параметр подключение к базе данных
db.url - параметр подключение к базе данных
db.username - параметр подключение к базе данных
db.password - параметр подключение к базе данных

captcha.public - ключ google recaptcha для  html страницы (публичный)
captcha.private - ключ google recaptcha для внутренней проверки (секретный)
